Bus Information Display

Relevant Skills

  • Python
  • Raspberry Pi
  • SolidWorks

Why?

The buses near my house ran inconsistently and I got tired of constantly checking my phone for live bus information. So I’d know when I had to leave to catch the bus, I made a desktop fixture to show arrival times.

About the Project

The device consists of a Raspberry Pi Zero W, an LCD screen, a toggle switch, and a 3D printed housing.

The Raspberry Pi runs a Python script which obtains bus information through Metro Vancouver’s TransLink API. The housing was designed in SolidWorks and 3D printed.
